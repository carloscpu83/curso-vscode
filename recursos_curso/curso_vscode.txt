#Tema 10 -> Comentar codigo
Comentario con // -> ctrl + mayus + /

#Tema 12 -> Creacion de archivos y carpetas
ctrl + click izq

#Tema 13 -> Ir a definicion
Saltar a la definicien -> ctrl + click izq = F12
Ver la definicion sin cambiar de documento -> F12

#Tema 14 -> Borrar lineas
Una o varias lineas seleccionadas -> ctrl + mayus + k
Seleccionar todas las lineas que tengan una ocurrencia y posteriormente borrarlas -> ctrl + mayus + l + k

#Tema 15 -> Deshacer y rehacer
Deshacer -> ctrl + z
Rehacer -> ctrl + mayus + z || ctrl + y

#Tema 16 -> Zen mode
Ocultar barra lateral -> ctrl + b
Zen mode -> (ctrl + k) + z

#Tema 17 -> Terminal
Mostrar y ocultar terminal -> ctrl + `

#Tema 18 -> Emmet wrap + atajos personalizados
Emmet wrap (encapsula code entre etiquetas) -> (ctrl + mayus + p) + escribir wrap + agregar el anidamiento de etiquetas (ejemplo: div>xx>zzz)

#Tema 19 -> Tabs
Cerrar actual -> ctrl + w
Cerrar todos -> ctrl + k + w
Ver todos los abiertos -> ctrl + tab
Reabrir -> ctrl + mayus + t
